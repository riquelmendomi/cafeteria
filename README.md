# Cafetería Lavka

Este proyecto consiste en una landing page totalmente responsive desarrollada para una cafetería ficticia llamada Café Lavka. La página está construida utilizando HTML semántico, Bootstrap 5 y JavaScript, integrando componentes como navegación, hero, carrusel, cards, formulario de reservas y un modal de confirmación.

La estructura está organizada siguiendo buenas prácticas, incluye validación básica de formulario y está versionada con Git. El despliegue final muestra un diseño limpio, moderno y adaptable a dispositivos móviles, tabletas y escritorio.
